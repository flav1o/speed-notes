# Speed Notes - Online Notepad

Simple online notepad where you can write your notes and send them to anyone you want or download the pad content to a .txt file. 
It's accountless and you can create a notepad with the name you want. You can also disable the line counter or change the background!

## How to use
Just enter a name you want for your notepad and you are good to go! If you want to send the notepad to someone just copy the pad name and send it to them!
